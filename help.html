Frequently Asked Questions

How do I import marks from SDMS/Urubuto?
Go to Classes → Select class → Click "Upload List" → Upload CSV/Excel file exported from Urubuto.
Can I use this software offline?
Yes! SmartClass Analytics works 90% offline. All data is stored locally.
How do I generate report cards?
Go to Reports → Select "Student Report Card" → Choose class and term → Generate → Print or save as PDF.
What are Smart Groups?
Smart Groups automatically creates balanced study groups by mixing top performers with students needing support.
