Frequently Asked Questions
📖 How do I import marks from SDMS/Urubuto?

Step-by-step guide:

  1. Go to Classes page from the sidebar
  2. Select the class you want to import marks for
  3. Click the "Upload List" button
  4. Choose the CSV/Excel file exported from SDMS/Urubuto
  5. Confirm the import - the system will automatically map the columns

Note: The file should contain student names and scores for the selected term.

💻 Can I use this software offline?

Yes! SmartClass Analytics works 90% offline.

  • All data is stored locally in your browser using localStorage
  • You can add classes, students, and marks without internet
  • Generate reports and view analytics offline
  • Only cloud backup and SDMS sync require internet connection

Your data remains on your device - you have full control!

📊 How do I generate report cards for students?

Generating report cards is easy:

  1. Navigate to the Reports page
  2. Select "Student Report Card" from the report type dropdown
  3. Choose the class and term (1, 2, or 3)
  4. Click "Generate Report"
  5. Use "Print" or "Export PDF" to save or print

You can also generate class summary reports and SDMS-compatible exports.

👥 What are Smart Groups and how do they work?

Smart Groups automatically creates balanced study groups:

  • Balanced Strategy: Mixes top performers with students needing support
  • Random Strategy: Creates random groups for variety
  • Best Together: Groups top performers together for advanced work

To use: Go to Smart Groups → Select a class → Choose number of groups → Pick strategy → Click "Generate Groups"

🔮 How accurate are the predictive insights?

Our AI model has 92% accuracy based on historical performance data.

  • Analyzes performance trends across multiple terms
  • Identifies students at risk of failing (declining performance)
  • Highlights students showing significant improvement
  • Provides confidence scores for each prediction
  • Gives actionable recommendations for intervention

Use these insights to provide targeted support to students who need it most!

💾 How do I backup and restore my data?

Backing up your data:

  1. Go to the Backup page
  2. Click "Export All Data" to download a JSON backup file
  3. Save the file to a safe location (external drive, cloud storage)

Restoring from backup:

  1. Click "Import Backup" on the Backup page
  2. Select your previously saved JSON file
  3. Confirm to restore - all data will be replaced

Pro tip: Enable Auto Backup to automatically save your data every day at 8:00 PM!

🎨 Can I customize the appearance?

Yes! Go to Settings page to customize:

  • Dark/Light Mode: Toggle between themes
  • Background Color: Choose your preferred background color
  • Text Color: Customize text appearance
  • Accent Color: Change the orange accent to any color you like

Changes apply to ALL pages instantly!

📱 Does this work on mobile phones?

Yes! SmartClass Analytics is fully responsive:

  • Works on smartphones, tablets, and desktop computers
  • Sidebar collapses into a hamburger menu on mobile
  • Tables scroll horizontally for easy viewing
  • Buttons are sized for touch input
  • All features work the same on any device!
