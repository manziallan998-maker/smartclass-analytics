Frequently Asked Questions

How do I import marks from SDMS/Urubuto?
Go to Classes → Select class → Click "Upload List" → Upload CSV/Excel file.
Can I use this offline?
Yes! All data is stored locally in your browser. Works 100% offline.
How do I generate report cards?
Go to Reports → Select "Student Report Card" → Choose class and term → Generate → Print.
What are Smart Groups?
Smart Groups automatically creates balanced study groups mixing top performers with students needing support.
How do I backup my data?
Go to Backup page → Click "Export All Data" to download JSON backup. Import to restore.
