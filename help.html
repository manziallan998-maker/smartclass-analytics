Frequently Asked Questions

How do I import marks from SDMS/Urubuto?
Go to Classes → Select your class → Click "Upload List" → Upload the CSV/Excel file exported from Urubuto. The system will automatically map the columns and import student marks.
Can I use this software offline?
Yes! SmartClass Analytics works 90% offline. All data is stored locally in your browser using localStorage. Only cloud backup and SDMS sync require internet connection.
How do I generate report cards for students?
Go to Reports → Select "Student Report Card" → Choose class and term → Click Generate → You can then export as PDF or print directly.
What is the Smart Groups feature?
Smart Groups automatically creates balanced study groups by mixing top performers with students needing support. You can choose grouping strategies: Balanced, Random, or Best Together.
How accurate are the predictive insights?
Our AI model has 92% accuracy based on historical performance data. It analyzes trends over multiple terms to predict which students may need intervention.
Can I export data for government reporting?
Yes! Use the "SDMS Compatible Export" in the Reports section. This generates files in the exact format required by Rwanda's Ministry of Education.
