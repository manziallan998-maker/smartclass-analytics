# SMARTCLASS ANALYTICS v2.0

## Teacher Performance Management System for Rwanda

### Features
- ✅ Teacher Authentication (Login/Signup)
- ✅ Class Management (Create, Edit, Delete)
- ✅ Student Management (Add, Edit, Delete, Import)
- ✅ Marks Entry (Term 1, 2, 3)
- ✅ Best/Low/Failing Student Filters
- ✅ Smart Groups (Balanced study groups)
- ✅ Performance Analytics (Charts & Graphs)
- ✅ Predictive Insights (AI-powered predictions)
- ✅ Lesson Planner with "Time to Leave" Popups
- ✅ Weekly Timetable View
- ✅ Report Generation (Class summary, Report cards, SDMS export)
- ✅ Competition Mode (Leaderboards & Badges)
- ✅ Backup & Restore (Export/Import all data)
- ✅ Global Settings (Colors affect ALL pages)
- ✅ Dark/Light Mode
- ✅ Offline First (Local Storage)
- ✅ Responsive Design (Works on phone/tablet/desktop)
